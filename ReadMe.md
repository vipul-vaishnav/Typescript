## Dev Env

**Code Editor:** VS Code

**Tools:** NodeJS, npm

## Installation

Install TypeScript with npm

```bash
  npm install -g typescript
```

To Check

```bash
  tsc -v
```

To Configure settings

```bash
  tsc --init
```
